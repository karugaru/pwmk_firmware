# ビルド手順

ビルド手順は利用する環境ごとに分けています。

- Visual Studio Code と Raspberry Pi Pico 拡張を使う場合: [ビルド手順_VSCode.md](ビルド手順_VSCode.md)
- ビルドスクリプトを使用する CLI ビルド: [ビルド手順_CLI.md](ビルド手順_CLI.md)

## VSCode

VSCodeを利用したビルド手順は、主にGUIを使用したビルド手順となります。VSCodeのRaspberry Pi Pico拡張がサポートする環境であれば、全て同じ手順でビルド可能です。

テスト済みの環境は以下の通りです。

- Windows 11
- Ubuntu 24.04

## CLI

CLIを利用したビルド手順は、Linux環境でのビルド手順となります。CLIビルドでは、ビルドスクリプトを使用して依存関係の導入からビルドまで自動化することが可能です。

テスト済みの環境は以下の通りです。

- Ubuntu 24.04
- Ubuntu 26.04
- Debian 11.11
- Debian 12.15
- AlmaLinux 9.8
- AlmaLinux 10.2
- Fedora 43
- Fedora 44
- Arch Linux (2026-07-23時点の最新パッケージでテスト済み)
